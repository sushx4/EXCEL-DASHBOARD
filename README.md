I recently created a complete interactive data analytics dashboard for Vrinda Store 2022 using Excel.
 This project helped me understand customer behavior, sales trends, state-wise performance, gender insights, and order channels in a detailed way.

 Key Insights:
Women contribute 64% of total sales
Top performing state: Uttar Pradesh
Teenagers have the highest order share (55%)
Highest sales through Amazon & Myntra
Most orders delivered successfully (92%)
